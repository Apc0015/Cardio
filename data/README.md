## Subdirectories

### `raw/`
Contains original datasets downloaded from Kaggle or other sources.
- **Do not modify** files in this directory
- Files are ignored by git (see `.gitignore`)
- Download using: `python scripts/download_data.py`

**Expected files:**
- `cardio_train.csv` - Original cardiovascular disease dataset

### `processed/`
Contains cleaned, engineered, and transformed datasets.
- Generated by running `notebooks/data_preprocessing.ipynb`
- Files are ignored by git

**Expected files:**
- `CVD_Cleaned.csv` - Cleaned and balanced dataset
- `train_data.csv` - Training split (80%)
- `test_data.csv` - Testing split (20%)

### `external/`
Contains external reference datasets or supplementary data.
- Medical reference ranges
- Population statistics
- Clinical guidelines data

## Getting Started

1. **Download Data:**
   ```bash
   python scripts/download_data.py
   ```

2. **Preprocess Data:**
   ```bash
   jupyter notebook notebooks/data_preprocessing.ipynb
   ```

3. **Verify Files:**
   ```bash
   ls -la data/raw/
   ls -la data/processed/
   ```

## Data Schema

### Raw Dataset
- **Records:** ~308,000
- **Features:** 19 original features
- **Target:** Cardiovascular disease (binary)

### Processed Dataset
- **Records:** ~567,000 (after SMOTE)
- **Features:** 27 engineered features
- **Balance:** 50/50 (balanced classes)

## Important Notes

- Large data files (`.csv`) are git-ignored
- Anyone cloning must download data separately
- Always use `data/raw/` for original data
- Save all processed data to `data/processed/`
- Document any external data sources
